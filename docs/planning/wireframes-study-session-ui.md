- **Success Green**: `oklch(0.65 0.2 140)` - Streak, positive metrics
- **Accent Orange**: `oklch(0.6 0.2 30)` - Highlights, warnings
- **Text Dark**: `oklch(0.3 0.15 250)` - Headings
